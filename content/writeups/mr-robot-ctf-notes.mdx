# Mr. Robot CTF notes

This TryHackMe room strengthened my recon habits: check exposed files, map the web app carefully, and let each discovery guide the next move.

## What I practiced

- Directory and file discovery from public hints.
- WordPress login review and controlled credential testing in a lab environment.
- Password cracking workflow with a provided wordlist.
- Shell handling and basic stabilization after initial access.
- SUID review for privilege escalation paths.

## What it taught me

The biggest lesson was patience. The path was not only about tools; it was about asking what the app had already revealed, then checking each trust boundary carefully.
