# RunCTF local challenge

RunCTF was a local boot-to-root practice challenge around virtual hosts, hidden hints, exposed files, and privilege escalation thinking.

## What I practiced

- Reading page source for hidden clues.
- Virtual host discovery.
- Directory discovery on the discovered host.
- Reviewing exposed configuration carefully.
- Looking for privilege paths after initial access.

## What it taught me

Small configuration leaks can become the whole path. For real systems, that means secrets, internal hostnames, and backup files need the same care as the main application.
