# Vercel Deployment Notes

1. Push this repository to GitHub.
2. Import the repository on Vercel.
3. Framework preset: Next.js.
4. Optional environment variable: `NEXT_PUBLIC_FORMSPREE_ENDPOINT` for hosted contact form submissions.
5. Build command: `npm run build`.
6. After deployment, test `/`, `/dashboard`, `/certifications`, `/hack-lab`, `/case-files/edumanage`, `/case-files/whatsupucc`, and `/opengraph-image`.
